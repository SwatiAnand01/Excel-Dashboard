# Sales Dashboard

An interactive Sales Dashboard built in Excel to analyze sales performance, customer behavior, regional trends, payment methods, and product categories. The dashboard provides dynamic visual insights using charts, KPIs, maps, and slicers for effective business analysis and decision-making.

---

## Key Features
- Interactive dashboard with slicers and filters
- KPI tracking for sales and delivery performance
- Region-wise sales analysis using India map
- Brand-wise and category-wise sales insights
- Monthly sales trend analysis
- Payment method comparison
- Return vs Non-return analysis

---

## Key KPIs
- Total Sales: ₹8,06,59,661
- Average Delivery Time: 5.5 Days
- Average Rating: 3
- Return Analysis Included
